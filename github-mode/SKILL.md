# GitHub Manager

Master GitHub operations with `gh` GitHub CLI for pull requests, issues, releases, and more.

## When to Use

- Creating and managing pull requests with `gh pr` commands
- Managing issues with `gh issue` commands
- Creating and managing releases
- Managing labels and repositories
- Working with GitHub projects
- Reviewing PRs locally with `gh pr checkout`
- Managing workflow runs and actions
- Fork and sync workflows

## Core Responsibilities

1. **Pull Request Management**: Create, review, merge, and manage PRs
2. **Issue Tracking**: Create, edit, and track issues
3. **Release Management**: Create and manage GitHub releases
4. **Repository Operations**: View, fork, sync repositories
5. **Project Management**: Manage GitHub Projects
6. **Label Management**: Create and manage labels
7. **Workflow Monitoring**: Monitor CI/CD runs
